# Medical & Travel Expense Request

## Project Overview

This project is developed using **HTML, CSS, and JavaScript** based on the provided Medical & Travel Expense Request document.

The application generates the expense request form dynamically and displays medical and travel expense details using different datasets. The main goal of this project is to recreate the document layout and handle dynamic data rendering.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)


## Project Structure

```
Medical-Travel-Expense-Request/
│── index.html
│── style.css
│── script.js
│── wcb-logo.jpeg
│── README.md
└── demo-video.mp4
```

---

## How to Run the Project

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Use the dataset buttons to switch between different data scenarios.
5. Click the print button to save the page as PDF.
